# Layout 容器布局

容器布局，可快速搭建页面常见结构，本组件底层采用 flex 布局，需要浏览器支持。在使用本组件进行页面布局时注意，mw-layout 的子组件必须是 mw-header,mw-sider,mw-content,mw-footer, 若使用嵌套结构，则 mw-layout 的子组件必须是 mw-layout,请在 mw-layout 的视图下进行布局操作。
